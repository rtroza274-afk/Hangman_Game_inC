Requirements to Build and Run Hangman Game (C)
================================================

Compiler:
- GCC (GNU Compiler Collection) or any standard C compiler
  - Windows: MinGW or MSYS2 recommended
  - Mac/Linux: GCC usually pre-installed

Standard Libraries Used (included with any C compiler):
- stdio.h
- stdlib.h
- string.h
- ctype.h
- time.h

Terminal Requirements:
- UTF-8 support recommended (for ASCII art banner to display correctly)

How to Compile:
    gcc main.c hangman.c -o hangman

How to Run:
    ./hangman        (Mac/Linux)
    hangman.exe      (Windows)
